# Security Policy
This repository contains university projects. Security updates are not applicable, but the code is maintained for learning purposes.

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability
If you notice any issues with the code (e.g. bugs/errors), you can report them by opening an issue in this repository.
1. Contact the project maintainer via mahimaahmed.tech@gmail.com
2. Include a clear description of the issue and steps to reproduce it.
3. You will receive updates about your report.
4. Accepted issues will be fixed promptly; declined reports will be explained.
